# 智能客服工作台 · 设计述职报告

## 部署到 GitHub Pages

1. 把这个目录里的所有文件推到 `banner-countdown-site` 仓库的 `main` 分支根目录
   ```
   index.html
   images/
   README.md
   ```
2. GitHub 仓库 → Settings → Pages
3. Source 选 `Deploy from a branch` → Branch 选 `main` / `(root)`
4. 保存，等 1-2 分钟

访问：`https://youzhaoyang.github.io/banner-countdown-site/`

## 本地预览

任意静态服务器都可以，例如：
```bash
npx serve .
```
